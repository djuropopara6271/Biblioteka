# Online Biblioteka – IT354

Web aplikacija za upravljanje online bibliotekom.  
Korisnici mogu da pregledaju i pozajmljuju knjige, dok administrator ima uvid u statistiku i upravljanje sadržajem.

---

## Tehnologije

- React (Vite)
- React Router
- Axios
- json-server
- Yup (validacija formi)
- Vitest (testiranje)

---

## Funkcionalnosti

- Pregled knjiga sa pretragom, sortiranjem i filtriranjem
- Pozajmljivanje knjiga (user)
- Admin panel za upravljanje knjigama
- Statistika pozajmica i korisnika
- Validacija formi
- Testovi komponenti

---

## Pokretanje projekta

```bash
npm install
npm run dev
npm run server
npm run test:run
```
